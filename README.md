# LumiLock Discord Bot

TypeScript 기반의 Discord 봇입니다. 현재 코드는 **열람 및 리뷰 목적**으로만 공개됩니다.

## 사용 권한 (중요)

- 허용: 코드 열람, 리뷰, 학습, 보안 평가.
- 금지: 무단 복제, 포크, 수정, 배포, 상업적/프로덕션 사용, 파생물 제작.  
  위 행위는 모두 **사전 서면 허락**이 필요합니다.

자세한 조건은 `LICENSE` 파일을 확인하세요.

## 개발

- 빌드: `npm run build`
- 개발 실행: `npm run dev`
- 프로덕션 실행: `npm run start`
